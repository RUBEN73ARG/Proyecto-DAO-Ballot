# Proyecto-DAO-Ballot

Aplicación Descentralizada (DApp) para un sistema de votación basado en el contrato inteligente estándar "Ballot" de Solidity.

## 🚀 Guía de Uso (Walkthrough)
Para entender cómo funciona el contrato, la lógica de votación y cómo interactuar con la interfaz, revisa nuestra guía detallada:

👉 **[Haz clic aquí para ver el Walkthrough del Proyecto](./walkthrough.md)**

## 🛠️ Tecnologías Utilizadas
- **Solidity**: Contrato inteligente Ballot.
- **Vite + React / TypeScript**: Interfaz de usuario del Frontend.
- **Ethers.js / Web3**: Conexión con la billetera blockchain.

## 📦 Instalación Local
Si quieres clonar este repositorio y ejecutarlo en tu computadora, sigue estos pasos en tu terminal:

1. Instala las dependencias necesarias:
   ```bash
   npm install
   ```
2. Inicia el servidor de desarrollo local:
   ```bash
   npm run dev
   ```
